# 📊 Análise de Vendas com Python

Este projeto consiste em uma análise exploratória de dados (EDA) focada em métricas de vendas. O objetivo é extrair insights estratégicos sobre faturamento, performance de produtos e distribuição regional para apoiar processos de tomada de decisão.

---

## 🛠️ Tecnologias e Bibliotecas
* **Python 3**: Linguagem principal.
* **Pandas**: Manipulação e tratamento de dados.
* **Matplotlib**: Visualização e geração de gráficos.

---

## 📂 Estrutura do Repositório
* `data/`: Contém os conjuntos de dados (arquivos CSV) utilizados na análise.
* `notebooks/`: Jupyter Notebooks contendo o código-fonte, processos de limpeza e análises detalhadas.

---

## 🔍 Escopo das Análises
A análise foi estruturada para responder a perguntas de negócio fundamentais:
* **Faturamento Global:** Consolidação da receita total no período.
* **Volume de Vendas:** Identificação dos produtos com maior giro de estoque.
* **Rentabilidade:** Rankeamento de produtos por contribuição financeira.
* **Performance Regional:** Distribuição do faturamento por zonas geográficas.

---

## 💡 Insights Estratégicos
A partir do processamento dos dados, foram identificados os seguintes pontos:
* **Liderança Regional:** A região **Sudeste** apresenta a maior concentração de faturamento.
* **Mix de Produtos:** O segmento de **Eletrônicos** é o principal motor de receita da operação.
* **Produto Estrela:** O **Celular** consolidou-se como o item de maior impacto financeiro no portfólio.
